# JS UTILS

This repo contains the javascript services that i use with my engine.  
  
## /fish
Lightly edited version of [ivangabriele/dockfish](https://github.com/ivangabriele/dockfish) to host the api on 0.0.0.0 instead of 127.0.0.1. Otherwise the exact same. Buildable stockfish 15 service image.
  

## /frontEnd
Very simple chess front end service for real time chess board updates. Accepts POST request to update the board and stats. Buildable service image as well.
